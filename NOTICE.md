# Notice

**Lendas contra Hordas** is an independent Windows game project.

This public repository intentionally contains only showcase material:

- selected screenshots and GIF previews;
- high-level gameplay description;
- public release notes;
- press-kit style information;
- links/placeholders for downloadable builds.

It intentionally does **not** include:

- full source code;
- internal production scripts;
- raw AI-generation prompts and source art sheets;
- balance spreadsheets or private design documents;
- unreleased mechanics, formulas, tuning tables, or private build tooling.

The goal is to let people see the project without making the complete game easy
to clone.
