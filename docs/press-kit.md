# Press Kit

## Nome

Lendas contra Hordas

## Gênero

RPG survivors-like / horde survival / action RPG.

## Plataforma

Windows.

## Descrição curta

Enfrente hordas crescentes, sobreviva a eventos mundiais, explore dungeons e
evolua classes lendárias em um RPG survivors-like de fantasia sombria.

## Descrição longa

**Lendas contra Hordas** combina combate rápido contra hordas com progressão
permanente, classes de fantasia e encontros de escala titânica. Cada run começa
controlável, mas cresce em pressão com monstros mais resistentes, eventos,
dungeons, drops, vendedores e escolhas de build.

O projeto busca a leitura clara de um survivors-like com uma direção visual de
pixel art moderna, usando personagens, monstros, VFX e HUDs desenhados para
funcionar mesmo com muitos elementos na tela.

## Pilares

- Classes com identidade: Guerreiro, Mago, Arqueiro e Invocador.
- Progressão permanente entre runs.
- Eventos mundiais e dungeons com recompensa.
- Combate escalável contra hordas.
- Monstros titânicos com presença visual maior.
